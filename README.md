# 📊 HR Workforce Insights & Employee Attrition Dashboard
## 🔍 Overview
This project presents a comprehensive **HR Analytics Dashboard** built using Power BI to analyze employee attrition, workforce trends, and key HR metrics.  
The dashboard enables HR teams and business leaders to make **data-driven decisions** for improving employee retention and workforce planning.

## 🎯 Objectives
- Analyze employee attrition across departments and job roles  
- Identify high-risk employees likely to leave  
- Track workforce trends over time  
- Evaluate factors like salary, satisfaction, and work mode  
- Provide actionable insights for HR decision-making  

## 🛠 Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query  
- Excel / CSV Dataset  

## 📂 Dataset Description
The dataset includes employee-level information such as:
- Employee ID, Name  
- Age, Gender, Department, Job Role  
- Monthly Income  
- Job Satisfaction & Work-Life Balance  
- Work Mode (Office / Hybrid / Remote)  
- Attrition Status & Risk Level  
- Joining Date  


## 🔄 Data Preparation (Power Query)
- Removed null and duplicate values  
- Converted data types  
- Created calculated columns:
  - Years at Company  
  - Age Groups  
  - Experience Bands  
- Extracted Year from Joining Date  


## 📐 Key DAX Measures
- Total Employees  
- Attrition Count  
- Attrition Rate (%)  
- Average Monthly Income  
- Average Job Satisfaction  
- High Risk Employees Count  
- Average Tenure
